# strategy.md

## 1인 기업 자동화 비즈니스 시스템 구현 명세서 (Auto-Pilot Business Spec)

### 1. 개요 (Project Overview)
본 문서는 1인 기업의 고질적인 문제인 '시간 부족'을 해결하고, 노동력 투입 없이 수익이 발생하는 **자율 비즈니스 모델(Autonomous Business Model)**을 구축하기 위한 기술 명세서이다. 네이버 스마트스토어 운영 효율화와 트래픽 생성을 위한 콘텐츠 자동화를 핵심 목표로 한다.

*   **목표:** 하루 2시간 미만 관리로 월 수익 시스템 유지 및 확장
*   **핵심 전략:** 노코드(No-Code) 툴과 AI 에이전트의 결합을 통한 '인지-추론-행동' 프로세스 자동화

---

### 2. 시스템 아키텍처 (System Architecture)

전체 시스템은 **중앙 제어(Orchestrator)**, **두뇌(AI Agent)**, **데이터베이스(DB)**, **실행 채널(Channel)**로 구성된다.

*   **Orchestrator (중앙 제어):** Make (구 Integromat) 또는 Zapier
*   **AI Engine (지능):** ChatGPT API (GPT-4o), Claude 3.5 Sonnet
*   **Database (저장소):** Google Sheets, Airtable
*   **Trigger (시작점):** RSS Feed, 시간 스케줄러, 웹훅(Webhook)

---

### 3. 모듈별 구현 명세 (Implementation Modules)

#### Module A. 커머스 오토파일럿 (Smart Store Automation)
판매자의 시간을 가장 많이 뺏는 물류와 CS, 정산 관리를 자동화한다.

**A-1. 물류 및 배송 자동화 (Logistics)**
*   **구현 논리:** 주문이 들어오면 판매자가 개입하지 않고 즉시 풀필먼트 센터로 데이터가 전송되어야 한다.
*   **프로세스:**
    1.  네이버 스마트스토어 API → 주문 데이터 수집 (10분 간격)
    2.  `Make` 시나리오 실행 → 주문 정보를 **네이버 풀필먼트 얼라이언스(NFA)** 파트너사(예: 파스토, 품고) 시스템으로 API 전송
    3.  송장 번호 생성 시 → 스마트스토어에 '발송 처리' 자동 업데이트
*   **핵심 도구:** Naver Commerce API, NFA 서비스

**A-2. AI 고객 응대 (CS Agent)**
*   **구현 논리:** 단순 반복 문의(배송 언제?, 재입고 언제?)는 AI가 처리하고, 복잡한 문제만 인간에게 넘긴다.
*   **프로세스:**
    1.  스마트스토어 '문의하기' 게시판/톡톡 메시지 수신 트리거
    2.  `ChatGPT API` 연동: 고객 질문 의도 분석 (배송/상품/환불 분류)
    3.  **RAG (검색 증강 생성):** 미리 업로드된 FAQ 및 상세 페이지 데이터를 참조하여 답변 생성
    4.  답변 자동 등록 및 고객에게 알림 발송
*   **참고:** 네이버 클로바(CLOVA) 라이브챗 활용 가능

**A-3. 수익 및 정산 시각화 (Financial Dashboard)**
*   **구현 논리:** 복잡한 수수료 구조를 자동 계산하여 순수익을 매일 리포팅한다.
*   **프로세스:**
    1.  `Google Apps Script` 실행 (매일 09:00)
    2.  스마트스토어 정산 내역 크롤링 또는 API 호출
    3.  수수료(결제 수수료, 매출 연동 수수료 등) 자동 차감 계산
    4.  슬랙(Slack) 또는 카카오톡으로 "어제 순수익: 000원" 알림 발송

---

#### Module B. 콘텐츠 팩토리 (Content Factory)
트래픽 유입을 위한 블로그 및 SNS 콘텐츠를 무한 생성한다.

**B-1. 정보성 블로그 포스팅 자동화 (SEO Targeting)**
*   **구현 논리:** 타겟 키워드(예: 디지털 마케팅, 건강)의 최신 트렌드를 읽고 재가공하여 포스팅한다.
*   **워크플로우 (Make 시나리오):**
    1.  **Source:** `Inoreader` RSS 피드에서 특정 키워드 뉴스 감지 (Watch Articles)
    2.  **Process:** `HTTP Request`로 본문 추출 → `Text Parser`로 HTML 태그 제거
    3.  **AI Generation:** `ChatGPT API`에 프롬프트 전송
        *   *Prompt:* "이 기사를 바탕으로 초보자가 이해하기 쉬운 블로그 글을 작성해줘. SEO 키워드는 OOO이고, 소제목을 포함해."
    4.  **Image:** `DALL-E 3`로 제목에 맞는 썸네일 자동 생성
    5.  **Action:** 워드프레스(WordPress) 또는 티스토리 API로 자동 발행 (Draft/Publish)

**B-2. 숏폼 영상 제작 자동화 (Short-form Automation)**
*   **구현 논리:** 텍스트나 이미지를 영상으로 변환하여 유튜브 쇼츠/릴스에 업로드한다.
*   **워크플로우:**
    1.  **Database:** Airtable에 여행지/상품 정보(텍스트, 이미지 URL) 저장
    2.  **Design:** `Placid` 또는 `Canva API`를 연동하여 이미지 템플릿에 텍스트/사진 자동 합성
    3.  **Video Gen:** 합성된 이미지들을 영상 편집 API(예: Shotstack)나 `Vrew` 등을 통해 영상화
    4.  **Action:** 유튜브/인스타그램 API를 통해 예약 업로드 (단, 유튜브 수익 창출 정책 고려하여 오리지널 콘텐츠 비중 유지 필요)

---

#### Module C. 리드 수집 및 판매 (Sales Funnel)
잠재 고객을 확보하고 유료 상품(전자책 등)으로 전환시킨다.

**C-1. 리드 마그넷 자동 발송**
*   **구현 논리:** 블로그에 이메일을 남기면 무료 전자책을 자동 발송한다.
*   **프로세스:**
    1.  `Google Forms` 또는 랜딩 페이지에서 이메일 제출
    2.  `Zapier` 트리거 발동
    3.  `Gmail` 또는 `Stibee` API를 통해 미리 준비된 PDF 파일 자동 발송
    4.  고객 DB를 구글 시트에 자동 저장하고 3일 뒤 후속 메일 예약

**C-2. 전자책 제작 파이프라인**
*   **구현 논리:** 축적된 블로그 글이나 노하우를 빠르게 상품화한다.
*   **프로세스:**
    1.  블로그 글 데이터 수집 (크롤링)
    2.  `ChatGPT` (Canvas 기능 활용)에게 목차 구성 및 챕터별 초안 작성 요청
    3.  `MS Word` 매크로 또는 `Canva` 일괄 생성 기능을 이용해 디자인 적용
    4.  크몽/검로드(Gumroad) 상세 페이지 업로드

### 4. 기술 스택 및 비용 (Tech Stack & Cost)

| 구분 | 도구(Tool) | 용도 | 예상 비용 (월) |
| :-- | :-- | :-- | :-- |
| **자동화 허브** | **Make (Integromat)** | 앱 간 연동 및 시나리오 실행 | 무료 ~ $9 (Core) |
| **AI 모델** | **ChatGPT API (OpenAI)** | 글쓰기, 이미지 생성, CS 답변 | 사용량 기반 (약 $5~$20) |
| **데이터베이스** | **Google Sheets / Airtable** | 주문 관리, 콘텐츠 소스 저장 | 무료 |
| **소스 수집** | **Inoreader** | 뉴스/트렌드 RSS 수집 | 무료 ~ $9 |
| **이미지/디자인** | **Placid / Canva** | SNS 이미지 템플릿 자동화 | 약 $19~39 |
| **플랫폼** | **WordPress / Naver** | 콘텐츠 발행 및 판매 | 호스팅비 약 $10 |

### 5. 실행 로드맵 (Action Plan)

1.  **1주차 (기획 및 환경 설정):** 판매 아이템 선정(위탁/전자책), 네이버 스마트스토어 개설, Make/OpenAI 계정 생성 및 API 키 발급.
2.  **2주차 (콘텐츠 자동화 구축):** RSS → GPT → 블로그 자동 포스팅 시나리오 구현 및 테스트 (오류 수정).
3.  **3주차 (커머스 연동):** 스마트스토어와 NFA 물류 시스템 연동, CS 자동 응답 메시지 세팅.
4.  **4주차 (확장 및 최적화):** SNS 자동화 추가, 정산 대시보드 구축, 유입 데이터를 분석하여 프롬프트 튜닝.

이 명세서는 '시간'을 '시스템'으로 대체하여 1인 기업이 24시간 수익을 창출할 수 있는 구체적인 청사진입니다. 초기 세팅에는 노력이 필요하지만, 구축 후에는 유지보수만으로 운영이 가능합니다.
